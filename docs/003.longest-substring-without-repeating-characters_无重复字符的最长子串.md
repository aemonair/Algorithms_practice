#### [3. 无重复字符的最长子串](https://leetcode.cn/problems/longest-substring-without-repeating-characters/)
中等

给定一个字符串 s ，请你找出其中不含有重复字符的 最长子串 的长度。

示例 1:

输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。

示例 2:

输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。

示例 3:

输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。
 
提示：

0 <= s.length <= 5 * 10^4
s 由英文字母、数字、符号和空格组成

---- ----

```cpp
	// 标记并删除umap
    int lengthOfLongestSubstring0(std::string s)
    {
        int start = 0;
        int end = 0;
        std::unordered_map<char, int> umap; // save counter
        int max = std::numeric_limits<int>::min();
        int size = s.size();
        for(end = 0; end < size; end++)
        {
            char c = s[end];
            if(umap.count(c) > 0)
            {
                while(umap.count(c) > 0 && start < end)
                {
                    umap.erase(s[start]);
                    start++;
                }
            }
            else
            {
                max = std::max (max, end - start + 1);
            }
            umap[c]++;
        }
        return max == std::numeric_limits<int>::min()? 0: max;
    }
```

```cpp
    int lengthOfLongestSubstring1(std::string s)
    {
        int start = 0;
        int end = 0;
        int size = s.size();
        int maxlength = 0;
        std::unordered_map<char, int> umap;//save greater position
        for(end = 0; end < size; end++)
        {
            char ch = s[end];
            if(umap.count(ch) > 0)
            {
                start = std::max(start, umap[ch]+1);
            }
            umap[ch] = end;
            maxlength = std::max(maxlength, end - start + 1);
        }

        return maxlength;
    }
```
